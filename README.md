# Ex1
This assignment introduces the use of functional programming and testing. The main task in this assignment is to implement a set of static functions over 1D arrays representing polynomials functions (for example: add,multiply,area,length,ect..).
In addition, the assignment includes writing JUnit tests class to test every function in the Ex1 class
